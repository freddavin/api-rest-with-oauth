# API REST with OAUTH

Projeto com objetivo de exercitar e aplicar alguns dos meus aprendizados de desenvolvimento de software para backend utilizando arquitetura MSC (Model, Service e Controller).

## Tecnologias utilizadas

- Node.js
- Typescript
- Express
- Mongoose
- Winston
- Node-Config
- Bcrypt
- Jsonwebtoken
